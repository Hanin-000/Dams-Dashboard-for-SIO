# Saudi Irrigation Organization – Dams Dashboard

## Overview

This project is an **interactive dashboard** designed to analyze and visualize dam data across Saudi Arabia. It provides users with a comprehensive view of dam distribution, types, purposes, and storage capacities in a clear and engaging visual format.

The dashboard was developed using **data analysis and visualization tools**, with a focus on delivering a smooth and intuitive experience for project managers, researchers, and decision-makers.

---

## Key Features

1. **Total Storage Capacity and Dam Count**

   * Displays the **total storage capacity (m³)** of all dams in Riyadh and Tabuk.
   * Shows the **total number of dams** in a clear overview.

2. **Interactive Dam Map**

   * Pinpoints the location of each dam on a map.
   * Allows users to view **detailed information for each dam**, including:

     * Dam name
     * Region and province
     * Dam image

3. **Statistics by Region/Province**

   * Shows the number of dams per region/province using a **bar chart**.
   * Allows sorting and filtering by province or region.

4. **Dam Type Analysis**

   * Categorizes dams into types such as:

     * Earthfill
     * Concrete
     * Rockfill
   * Displays the count of each type visually.

5. **Dam Purpose Analysis**

   * Represents the purpose of dams using a **pie chart**, including:

     * Control
     * Compensation
     * Protection

6. **Additional Statistics**

   * Displays the number of dams by year of construction to track projects over time.
   * Highlights the **top 10 tallest dams** in the Kingdom with visual comparison.

7. **Interactive Data Filtering**

   * Users can select a specific region using a **dropdown filter**.
   * All charts and visualizations update dynamically based on the selection.

8. **Clear and Professional Visual Design**

   * Consistent color scheme and easy-to-read visuals.
   * Includes dam images for enhanced clarity and interaction.

---

## Technologies Used

* **Tableau** – for interactive visualizations and dashboards.
* **Excel** – raw data source for dam information; data was extracted from PowerPoint and organized in Excel.

---

## Project Goals

* Provide a **comprehensive overview of all dams** in Riyadh and Tabuk.
* Assist decision-makers in **quickly analyzing key data**.
* Support **future planning of dams and water projects**.
* Deliver **interactive and shareable visual reports**.
